# 🧠 Jogos de Linguagem – Ludwig Wittgenstein

## 📌 Sobre o Projeto
Este trabalho aborda o conceito de **Jogos de Linguagem**, desenvolvido por Ludwig Wittgenstein, e sua aplicação no ensino e na Programação Orientada a Objetos.

---

## 📖 1. Conceito

Os **Jogos de Linguagem** são uma teoria filosófica que afirma que o significado das palavras depende do seu uso em diferentes contextos.

Ou seja, a linguagem funciona como um "jogo", onde cada situação tem suas próprias regras.

---

## 🎓 2. Jogos de Linguagem no Ensino

No ensino, essa ideia mostra que aprender não é apenas decorar conceitos, mas saber utilizá-los na prática.

### ✔ Aplicações:
- Aprendizagem contextualizada
- Participação ativa do aluno
- Melhor compreensão dos conteúdos

---

## 💻 3. Jogos de Linguagem na Programação (POO)

Na programação, termos como:

- Classe
- Objeto
- Herança
- Método

possuem significados específicos dentro do contexto da Programação Orientada a Objetos.
