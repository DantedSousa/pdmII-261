# Atividade 03

## 📌 Descrição

Completar o código disponível em:
https://github.com/ricdtaveira/mobdev-parte-01/blob/master/14-poo/14-agregacao.dart

## 📝 Objetivo

Implementar no método `main()`:

1. Criar vários objetos Dependentes
2. Criar vários objetos Funcionário
3. Associar os Dependentes criados aos respectivos funcionários
4. Criar uma lista de Funcionários (Coleção Lista)
5. Criar um objeto Equipe Projeto chamando o método construtor que dá nome ao projeto e insere uma coleção de funcionários
6. Printar no formato JSON o objeto Equipe Projeto

## 💻 Tecnologias

* Dart
